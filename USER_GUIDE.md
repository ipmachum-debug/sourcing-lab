## 1. 시작하기

### 1-1. 회원가입 & 로그인

1. **https://lumiriz.kr** 에 접속합니다.
2. **회원가입**: 우측 하단 "회원가입" 링크를 클릭하여 이메일, 이름, 비밀번호를 입력합니다.
3. **관리자 승인**: 회원가입 후 관리자 승인을 기다립니다. 승인 전에는 "승인 대기" 화면이 표시됩니다.
4. **로그인**: 승인이 완료되면 이메일과 비밀번호로 로그인합니다.
5. **비밀번호 찾기**: 로그인 화면에서 "비밀번호 찾기"를 클릭하여 비밀번호를 재설정할 수 있습니다.

### 1-2. 화면 구성

로그인하면 좌측에 **사이드바 메뉴**, 우측에 **콘텐츠 영역**이 표시됩니다.

| 메뉴 | 설명 |
|------|------|
| 🏠 대시보드 | 전체 현황 요약 |
| 📝 데일리 소싱 | 매일 소싱 상품 등록 |
| 💰 Daily Profit | 일별/주별/월별 수익 관리 |
| 🛍️ 쿠팡 관리 | 쿠팡 API 연동, 매출/주문/정산 |
| 🐢 소싱 헬퍼 | Chrome 확장프로그램 데이터 허브 |
| 📊 헬퍼 대시보드 | 확장프로그램 상세 분석 대시보드 |
| 📦 확장프로그램 | 설치 가이드 & 다운로드 |
| 📦 전체 상품 | 등록된 모든 상품 목록 |
| 🧪 테스트 후보 | 85점 이상 자동 분류된 상품 |
| 📅 주간 리뷰 | 매주 소싱 활동 리뷰 |
| 👤 내 프로필 | 프로필 정보 수정 |
| ⚙️ 계정 설정 | 계정 관련 설정 |

> **팁**: 사이드바 좌측 상단의 `≡` 버튼으로 사이드바를 접을 수 있습니다. 사이드바 우측 가장자리를 드래그하여 너비를 조절할 수도 있습니다.

## 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 이번 주의 소싱 현황을 한눈에 파악할 수 있습니다.

### 2-1. 요약 카드 (상단)

| 카드 | 내용 | 클릭 시 이동 |
|------|------|-------------|
| **이번 주 소싱** | 이번 주에 등록한 상품 수 | 데일리 소싱 |
| **테스트 후보** | 85점 이상 자동 분류된 상품 수 | 테스트 후보 |
| **이번 주 평균** | 이번 주 소싱 상품의 평균 점수 (100점 만점) | - |
| **주간 리뷰** | 이번 주 주간 리뷰 작성 여부 (금요일까지) | 주간 리뷰 |

### 2-2. 쿠팡 판매 현황

- 쿠팡 API가 연동되어 있으면 **오늘 주문 수, 판매 수량, 월 매출, 월 정산** 정보가 표시됩니다.
- **"설정 필요"** 배지가 보이면 쿠팡 관리 페이지에서 API 키를 등록해야 합니다.
- 카드를 클릭하면 쿠팡 관리 페이지로 이동합니다.

### 2-3. 판매 수익 현황

- **일간 / 주간 / 월간** 탭으로 전환하여 판매수량, 매출, 순이익을 확인합니다.
- 이익률이 20% 이상이면 핑크색, 10% 이상이면 보라색 배지로 표시됩니다.
- "Daily Profit Board 바로가기"를 클릭하면 상세 수익 관리 페이지로 이동합니다.

### 2-4. 고득점 상품 TOP 5 & 최근 등록 상품

- **고득점 상품 TOP 5**: 점수가 높은 상위 5개 상품이 표시됩니다. S등급(90+), A등급(80+) 등으로 분류됩니다.
- **최근 등록 상품**: 가장 최근에 등록한 상품 목록입니다.
- 각 상품을 클릭하면 상품 상세 페이지로 이동합니다.

### 2-5. 이번 주 카테고리 & 키워드

- **카테고리**: 이번 주 소싱한 상품의 카테고리별 분포
- **키워드 TOP**: 이번 주 가장 많이 사용한 키워드

## 3. 데일리 소싱

매일 소싱한 상품을 체계적으로 기록하는 페이지입니다.

### 3-1. 기본 정보 입력

| 항목 | 설명 | 필수 |
|------|------|------|
| 날짜 | 소싱한 날짜 (기본: 오늘) | O |
| 카테고리 | 생활용품, 주방용품, 전자기기 등 19개 카테고리 | - |
| 상품명 | 소싱하려는 상품의 이름 | **O** |
| 키워드 1~3 | 핵심 키워드, 보조 키워드, 롱테일 키워드 | **O** |
| 타겟 고객 | 예: "30대 여성", "자취생" 등 | - |
| 시즌성 | 연중, 여름, 겨울, 명절 등 | - |

### 3-2. 경쟁 & 차별화 평가

| 항목 | 선택지 | 점수 |
|------|--------|------|
| **경쟁도** | 낮음(20점) / 보통(12점) / 높음(6점) / 매우높음(2점) | 최대 20점 |
| **차별화 가능성** | 높음(20점) / 보통(12점) / 낮음(4점) | 최대 20점 |

### 3-3. 분석 & 메모

| 항목 | 내용 | 점수 |
|------|------|------|
| 썸네일 분석 | 경쟁사 썸네일 특징, 차별화 포인트 | 5점 |
| 상세페이지 포인트 | 상세페이지 구성, 강조할 내용 | 5점 |
| 증정품 아이디어 | 사은품, 증정품 아이디어 | - |
| **단점 보완 방안** | 경쟁사 단점 분석 및 보완 전략 | **8점** |
| **개발 노트** | 상품 개발 방향, OEM/ODM 포인트 | **7점** |
| 최종 의견 | 종합 판단, 진행 여부 | 5점 |

### 3-4. 실시간 점수 시스템

우측 패널에서 입력 내용에 따라 **실시간으로 점수가 계산**됩니다.

| 점수 | 등급 | 자동 상태 |
|------|------|----------|
| 90~100 | **S** | 테스트 후보 |
| 80~89 | **A** | 테스트 후보 |
| 65~79 | **B** | 검토중 |
| 50~64 | **C** | 보류 |
| 0~49 | **D** | 초안 |

> **85점 이상**이면 자동으로 "테스트 후보"로 분류됩니다!

### 3-5. 참고 링크

- **쿠팡 링크**: 해당 상품의 쿠팡 URL
- **참고 링크**: 1688.com 등 소싱처 URL
- 저장 후 상품 상세 페이지의 "키워드링크" 탭에서 키워드별 상세 링크(쿠팡 10개 + 1688 10개)를 관리할 수 있습니다.

### 3-6. 저장

- 우측 상단 또는 점수 패널 하단의 **"저장"** 버튼을 클릭합니다.
- "초기화" 버튼으로 모든 입력을 리셋할 수 있습니다.

## 4. 전체 상품 관리

등록된 모든 소싱 상품을 목록으로 관리하는 페이지입니다.

### 4-1. 상품 목록

- 상품명, 카테고리, 점수/등급, 상태, 등록일이 표시됩니다.
- 상품을 클릭하면 **상품 상세 페이지**로 이동합니다.

### 4-2. 상품 상태

| 상태 | 의미 |
|------|------|
| 초안 | 기본 등록 상태 |
| 검토중 | 추가 검토가 필요한 상품 |
| 테스트후보 | 85점 이상 자동 분류 |
| 테스트중 | 실제 테스트 진행 중 |
| 보류 | 보류 상태 |
| 폐기 | 소싱 포기 |
| 선정 | 최종 선정된 상품 |

### 4-3. 상품 상세 페이지

상품을 클릭하면 다음 정보를 확인/수정할 수 있습니다:
- 기본 정보 (상품명, 카테고리, 키워드 등)
- 경쟁/차별화 평가
- 분석 메모
- 키워드별 링크 관리 (쿠팡/1688)
- 경쟁사 정보
- 공급처 정보
- 마진 시뮬레이션
- 메모

## 7. Daily Profit Board (수익 관리)

상품별 일일 수익을 관리하는 페이지입니다.

### 7-1. 보기 모드

| 모드 | 설명 |
|------|------|
| **일간** | 특정 날짜의 상품별 판매량, 매출, 수익 |
| **주간** | 한 주간의 합산 데이터 |
| **월간** | 한 달간의 합산 데이터 |
| **연간** | 한 해 전체 합산 데이터 |

### 7-2. 일간 모드 기능

1. **날짜 선택**: `◀` `▶` 버튼으로 날짜를 이동하거나, 날짜를 직접 선택합니다.
2. **상품별 입력**:
   - 판매가, 마진, 수량을 입력하면 매출과 수익이 자동 계산됩니다.
   - 메모를 추가할 수 있습니다.
3. **저장**: 변경사항을 저장합니다.

### 7-3. 요약 카드

상단에 **총 판매량**, **총 매출**, **총 순이익** 요약 카드가 표시됩니다.

### 7-4. CSV 다운로드

데이터를 CSV 파일로 내려받을 수 있습니다.

## 8. 쿠팡 관리

쿠팡 오픈 API를 연동하여 매출, 주문, 정산 데이터를 자동으로 수집하고 관리하는 페이지입니다.

### 8-1. 쿠팡 계정 등록

1. **"+ 계정 추가"** 버튼을 클릭합니다.
2. 쿠팡 WING에서 발급받은 **Access Key**와 **Secret Key**를 입력합니다.
3. 벤더 ID를 입력합니다.
4. **"저장"**을 클릭하면 API 연결을 자동으로 테스트합니다.

### 8-2. 탭별 기능

| 탭 | 기능 |
|----|------|
| **대시보드** | 매출 요약, 주문 현황, 정산 현황 |
| **상품 매핑** | 소싱 상품 ↔ 쿠팡 상품 매핑 관리 |
| **매출/주문** | 기간별 주문 데이터 조회, 동기화 |
| **리포트** | 기간별 매출 리포트 |
| **설정** | API 키 관리, 동기화 설정 |

### 8-3. 데이터 동기화

- **수동 동기화**: "동기화" 버튼을 클릭하여 최신 데이터를 가져옵니다.
- 기간을 선택하여 특정 기간의 데이터를 동기화할 수 있습니다.
- 동기화 이력에서 성공/실패 여부와 수집된 건수를 확인할 수 있습니다.

### 8-4. 상품 매핑

소싱 상품과 쿠팡에서 실제 판매 중인 상품을 연결(매핑)하여 매출 추적을 합니다.

1. "상품 매핑" 탭에서 소싱 상품 선택
2. 쿠팡 상품 ID를 입력하여 매핑
3. 매핑된 상품의 판매 데이터가 자동으로 연결됩니다.

## 9. 소싱 헬퍼 (Chrome 확장프로그램)

쿠팡 검색 결과를 자동 분석하는 Chrome 확장프로그램입니다.

### 9-1. 설치 방법

1. 사이드바의 **📦 확장프로그램** 메뉴를 클릭합니다.
2. **"다운로드 v5.7.0"** 버튼을 클릭하여 ZIP 파일을 받습니다.
3. 다운로드한 ZIP 파일의 **압축을 해제**합니다.
4. Chrome 주소창에 **`chrome://extensions`** 입력 후 이동합니다.
5. 우측 상단의 **"개발자 모드"** 토글을 켭니다.
6. **"압축해제된 확장 프로그램을 로드합니다"** 클릭 → 압축 해제한 폴더를 선택합니다.
7. 쿠팡 사이트에 접속 후, Chrome 우측 상단의 **퍼즐 아이콘** → **"Coupang Sourcing Helper"** 클릭으로 사이드 패널을 엽니다.

> **팁**: 퍼즐 아이콘 옆의 📌 핀 버튼을 클릭하면 도구 모음에 항상 표시됩니다.

### 9-2. 확장프로그램 주요 기능

#### (1) 분석 탭 — 경쟁도 분석

쿠팡에서 키워드를 검색하면 **자동으로 상위 36개 상품**을 분석합니다.

- **경쟁 강도 (0~100점)**: 평균 리뷰수, 리뷰100+ 비율, 평균 평점, 광고 비율로 자동 계산
  - 🟢 **약함** — 소싱 기회!
  - 🟡 **보통** — 진입 가능
  - 🔴 **강함** — 차별화 필요

- **소싱 점수 (A~F 등급)**: 각 상품의 소싱 난이도 평가
  - **A (80+)**: 매우 좋음 — 소싱 강력 추천
  - **B (65+)**: 좋음 — 소싱 추천
  - **C (50+)**: 보통 — 검토 필요
  - **D (35+)**: 어려움 — 신중히 판단
  - **F (35미만)**: 매우 어려움 — 비추천

- **필터 & 정렬**:
  - 광고 제외, 소싱쉬운것(60점 이상), 상위 N개(5/10/20/36개), 정렬(가격/리뷰/평점/소싱점수)

#### (2) 후보 탭 — 소싱 후보 관리

- **⭐ 후보 저장**: 분석 탭에서 ⭐ 버튼을 누르면 후보로 저장 (최대 500개)
- **🔍 1688 찾기**: 상품 제목에서 키워드를 자동 추출하여 1688.com에서 소싱처 검색
- **삭제**: 불필요한 후보 정리

#### (3) 순위 탭 — 키워드 순위 추적

1. 추적할 검색 키워드와 타겟 상품 ID를 입력 후 "추가"
2. 해당 키워드로 쿠팡에서 검색할 때마다 **자동으로 순위 기록**
3. "📊 보기" 버튼으로 순위 확인 (타겟 상품은 보라색으로 강조)

#### (4) 상세 탭 — 상품 상세 분석

쿠팡 상품 상세 페이지를 열면 자동으로 파싱:
- 판매자명, 카테고리 경로, 원래가격/할인율
- 로켓배송/무료배송, 옵션 수, 구매 건수
- 평점/리뷰 수, 가격/리뷰 변동 이력

> 같은 상품을 여러 번 방문하면 가격과 리뷰 변화를 시간순으로 확인할 수 있습니다.

#### (5) 마진 탭 — 마진 계산기

1688 원가에서 쿠팡 판매가 기준으로 예상 순이익과 마진율을 계산합니다.

| 입력 항목 | 설명 | 기본값 |
|-----------|------|--------|
| 1688 원가 (CNY) | 상품 원가 (위안) | - |
| 환율 (CNY→KRW) | 위안→원 환율 | 190 |
| 국제배송비 (원) | 1건당 배송비 | 3,000원 |
| 관부가세율 (%) | 관세+부가세 | 10% |
| 쿠팡 판매가 (원) | 예상 판매가 | - |
| 쿠팡 수수료율 (%) | 카테고리별 수수료 | 10.8% |

**결과**: 원가(KRW), 배송비, 관부가세, 총 원가, 수수료, **순이익 / 마진율**
- 🟢 초록색: 이익
- 🔴 빨간색: 손해

#### (6) 서버 탭 — 서버 연동

lumiriz.kr 계정으로 로그인하면 모든 데이터가 서버에 자동 동기화됩니다.

동기화되는 데이터:
- ✅ 검색 스냅샷 (검색어, 상품 목록, 통계)
- ✅ 소싱 후보 목록 및 상태
- ✅ 순위 추적 데이터
- ✅ 상품 상세 가격/리뷰 변동

### 9-3. 업데이트 방법

### 10-2. 대시보드 탭 (개요)

- **최근 7일 활동**: 검색 분석, 후보 저장, 순위 기록, 상세 파싱 횟수
- **통계 카드**: 총 검색, 검색어 수, 평균 경쟁도, 저장 후보, 평균 소싱점수, 추적 키워드
- **검색 트렌드 미니 차트**: 30일간 검색 횟수 변화
- **AI 소싱 추천 미리보기**: 추천 키워드/상품
- **AI 리뷰 분석 미리보기**: 최근 분석 결과 요약
- **검색 수요 TOP 5**: 종합점수 기준 상위 키워드
- **자주 검색한 키워드 TOP 10**
- **후보 상태별 현황**

### 10-3. 검색 수요 탭

쿠팡 검색 데이터를 기반으로 **키워드별 수요와 경쟁도**를 분석합니다.

#### 사용 방법:
1. 쿠팡에서 관심 키워드를 검색합니다 (확장프로그램이 자동 저장).
2. **"통계 계산"** 버튼을 클릭하여 일별 통계를 생성합니다.
3. 좌측 키워드 목록에서 키워드를 선택하면 우측에 **일별 추이 그래프**가 표시됩니다.

#### 주요 지표:

| 지표 | 설명 |
|------|------|
| **상품수** | 해당 키워드의 검색 결과 상품 수 |
| **평균가** | 상위 상품들의 평균 가격 |
| **리뷰 증가** | 전일 대비 리뷰 증가량 |
| **판매 추정** | 리뷰 증가량 x 20 (리뷰 1개 = 판매 15~25건) |
| **경쟁도** | 0(블루오션) ~ 100(레드오션) |
| **수요 점수** | 리뷰 증가량 기반 수요 점수 |
| **종합 점수** | 리뷰증가 x 0.5 + 상품당리뷰 x 30 + (1-광고비율) x 20 |

#### 키워드 관리:
- **개별 삭제**: 각 키워드 옆 🗑️ 버튼
- **일괄 삭제**: 체크박스로 선택 후 "N개 삭제" 버튼
- **정렬**: 종합점수, 수요점수, 리뷰증가, 판매추정, 경쟁도, 평균가 기준

### 10-4. AI 인사이트 탭

축적된 데이터를 기반으로 AI가 분석한 인사이트를 제공합니다.

- **데이터 축적 현황**: 축적 기간, 추적 키워드 수, 총 스냅샷 수
- **놓친 기회**: 낮은 경쟁도, 광고 기회, 고마진 기회 등
- **파생 상품 제안**: 기존 키워드에서 파생 가능한 새로운 상품 아이디어
- **경쟁자 동향**: 리뷰 급증, 가격 변동 등 경쟁자 변화 알림

> **중요**: 매일 1~2회 관심 키워드를 검색하여 데이터를 축적할수록 분석이 정확해집니다.

### 10-5. 소싱 후보 관리 탭

확장프로그램에서 ⭐ 저장한 소싱 후보를 관리합니다.

- **상태 필터**: 신규, 검토중, 공급처 연락, 샘플 주문, 탈락, 선정
- **상태 변경**: 드롭다운으로 상태를 변경합니다.
- **소싱 상품 승격**: "승격" 버튼으로 데일리 소싱 상품으로 등록합니다.
- **CSV 내보내기**: 후보 데이터를 CSV 파일로 다운로드합니다.
- **쿠팡 링크**: 원본 쿠팡 상품 페이지로 바로 이동합니다.

### 10-6. 내 상품 추적 탭 (v5.7.0 NEW)

소싱 중인 상품의 **가격, 순위, 리뷰, 경쟁자 변동을 자동 추적**합니다.

#### 자동 등록:
- 소싱 상품, 후보, 쿠팡 매핑에서 자동으로 추적 상품이 등록됩니다.
- **"자동 등록"** 버튼을 클릭하면 일괄 등록됩니다.

#### 추적 데이터:
- **일일 스냅샷**: 가격, 리뷰 수, 순위, 경쟁자 수가 매일 기록됩니다.
- **변동 감지**: 가격 5%+ 변동, 순위 3위+ 변동 시 자동 알림
- **AI 분석**: 가격 추세, 리뷰 동향, 순위 변동, 경쟁자 비교, 파생 키워드 제안

#### 상세 보기:
상품을 클릭하면:
- 가격/순위/리뷰 추이 그래프
- 경쟁 상품 비교
- AI 제안 내용

### 10-7. 순위 추적 탭

키워드별 상품 순위를 추적합니다.

1. 좌측에서 **추적 키워드**를 선택합니다.
2. 우측에 **최신 순위** 테이블이 표시됩니다.
   - 순위, 상품명, 가격, 리뷰, 평점
   - AD(광고), 🚀(로켓배송) 표시
   - 타겟 상품은 보라색 배경으로 강조
3. **순위 변동 히스토리**: 7/14/30일 기간별 히스토리
4. **순위 트렌드 차트**: 특정 상품의 순위 변화를 라인 차트로 확인

> 히스토리 테이블에서 상품 행을 클릭하면 해당 상품의 트렌드 차트가 표시됩니다.

### 10-8. AI 리뷰 분석 탭 (GPT-4o)

OpenAI GPT-4o를 활용한 시장 분석입니다.

#### 사용 방법:
1. 입력창에 **분석할 키워드**를 입력합니다.
2. **"AI 분석 실행"** 버튼을 클릭합니다.
3. GPT가 해당 키워드의 상위 상품 데이터를 분석하여 결과를 제공합니다.

#### 분석 결과 항목:

| 항목 | 내용 |
|------|------|
| **시장 개요** | 분석 상품 수, 경쟁도, 평균가, 광고 비율, 로켓배송 비율, 리뷰100+ 비율 |
| **요약** | 시장 전체 요약 |
| **소싱 기회** | potential이 high/medium/low인 기회 목록 |
| **주의 사항** | severity가 high/medium/low인 위험 요소 |
| **고객 니즈** | 고객이 원하는 기능/품질 분석 |
| **추천 액션** | 구체적인 소싱/판매 전략 제안 |

| **부정적 요소** | 주의해야 할 부정적 신호 |
| **품질 우려사항** | 품질 관련 주의점 |

## 12. 자주 묻는 질문 (FAQ)

### Q. Chrome 확장프로그램은 Chrome 웹 스토어에 없나요?
**A.** 아직 Chrome 웹 스토어에 출시되지 않았습니다. ZIP 파일을 다운로드한 후 "압축해제된 확장 프로그램 로드" 방식으로 설치해주세요.

### Q. 확장프로그램이 쿠팡 이외의 사이트에서도 동작하나요?
**A.** 아니요. 쿠팡 검색 결과 페이지(`coupang.com`)와 상품 상세 페이지, 그리고 WING 셀러센터에서만 동작합니다.

### Q. 데이터는 어디에 저장되나요?
**A.** 확장프로그램에서 서버 탭으로 lumiriz.kr에 로그인하면 서버에 안전하게 저장됩니다. 로그인하지 않으면 브라우저 로컬 저장소에만 저장됩니다.

### Q. AI 리뷰 분석에 비용이 발생하나요?
**A.** AI 분석은 **무료**로 제공됩니다. OpenAI GPT-4o 기반으로 동작하며, API 장애 시 규칙 기반 분석으로 자동 대체됩니다.

### Q. 확장프로그램 업데이트는 어떻게 하나요?
**A.** 새 버전 ZIP을 다운로드한 후, `chrome://extensions`에서 기존 확장프로그램을 제거하고 새로 로드하면 됩니다.

### Q. 소싱 점수가 85점 이상이면 어떻게 되나요?
**A.** 자동으로 "테스트 후보"로 분류됩니다. 테스트 후보 페이지에서 모아서 볼 수 있습니다.

### Q. 쿠팡 API 키는 어디서 발급받나요?
**A.** 쿠팡 WING 셀러센터(wing.coupang.com)에서 발급받을 수 있습니다. WING > 개발자센터 > API 관리에서 Access Key와 Secret Key를 확인하세요.

### Q. 검색 수요 추정의 "판매 추정"은 정확한가요?
**A.** 리뷰 증가량 x 20으로 계산됩니다 (리뷰 1개당 약 15~25건 판매 추정). 정확한 수치는 아니지만, 키워드 간 상대적 비교에 유용합니다. 데이터가 축적될수록 정확도가 높아집니다.

### Q. 내 상품 추적은 어떻게 동작하나요?
**A.** 소싱 상품, 후보, 쿠팡 매핑에서 자동으로 추적 상품이 등록됩니다. 해당 키워드로 쿠팡 검색 시 경쟁자/유사상품/순위 데이터가 자동 수집되며, 가격 5%+ 변동이나 순위 3위+ 변동 시 알림이 생성됩니다.
