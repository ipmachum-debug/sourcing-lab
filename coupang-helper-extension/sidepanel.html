          쿠팡 소싱을 더 스마트하게! 설치부터 활용까지 한눈에 알아보세요.

          🔧

          1. 설치 방법

          ▼
          1
          확장프로그램 파일 다운로드

          coupang-helper-extension-v4.2.zip 파일을 다운로드하고 압축을 풀어주세요.

          2
          Chrome 확장프로그램 페이지 열기

          Chrome 브라우저 주소창에 chrome://extensions 입력 후 이동합니다.

          3
          개발자 모드 활성화

          우측 상단의 "개발자 모드" 토글을 켜세요.

          4
          압축해제된 확장 프로그램 로드

          좌측 상단 "압축해제된 확장 프로그램을 로드합니다" 클릭 → 압축 해제한 폴더를 선택합니다.

          5
          사이드 패널 열기

          Chrome 우측 상단의 퍼즐 아이콘 → "Coupang Sourcing Helper" 클릭으로 사이드 패널을 엽니다.

          📊

          2. 분석 탭 — 경쟁도 분석

          ▼

          쿠팡에서 검색하면 자동으로 상위 36개 상품을 분석합니다.

          🎯
          경쟁 강도 (0~100)

          평균 리뷰수, 리뷰100+ 비율, 평균 평점, 광고 비율로 자동 계산됩니다.

          약함 (소싱 기회!) 보통 (진입 가능) 강함 (차별화 필요)
          ⭐
          소싱 점수 (A~F)

          각 상품마다 리뷰, 평점, 가격 비율, 광고 여부, 로켓배송을 종합하여 소싱 난이도를 평가합니다.

          A (80+)매우 좋음
          B (65+)좋음
          C (50+)보통
          D (35+)어려움
          F (35미만)매우 어려움
          🔍
          필터 & 정렬

          광고 제외, 소싱 쉬운것만 보기(점수 60+), 상위 N개 선택, 가격/리뷰/평점/소싱점수 정렬이 가능합니다.

          ⭐

          3. 후보 탭 — 소싱 후보 관리

          ▼
          💾
          후보 저장

          분석 탭에서 ⭐ 버튼을 누르면 후보로 저장됩니다. 최대 500개까지 저장 가능합니다.

          🔍
          1688 찾기

          상품 제목에서 핵심 키워드를 자동 추출하여 1688.com에서 유사 중국 소싱처를 검색합니다.

          🗑
          후보 삭제

          불필요한 후보는 삭제 버튼으로 정리할 수 있습니다.

          📈

          4. 순위 탭 — 키워드 순위 추적

          ▼

          특정 키워드에서 내 상품의 순위 변화를 추적할 수 있습니다.

          1
          키워드 등록

          순위 탭에서 추적할 검색 키워드와 (선택) 타겟 상품 ID를 입력 후 "+ 추가"를 누릅니다.

          2
          자동 기록

          쿠팡에서 해당 키워드를 검색할 때마다 자동으로 순위가 기록됩니다.

          3
          순위 확인

          "📊 보기" 버튼으로 최신 순위를 확인하세요. 타겟 상품은 보라색으로 강조됩니다.

          💡 분석 탭에서 상품 옆의 📈 버튼으로도 바로 순위 추적을 등록할 수 있습니다.
          🔍

          5. 상세 탭 — 상품 상세 분석

          ▼

          쿠팡 상품 상세 페이지(예: coupang.com/vp/products/12345)를 열면 자동으로 파싱합니다.

          📋
          파싱되는 정보
          • 판매자명, 카테고리 경로
          • 원래가격, 할인율, 현재가격
          • 로켓배송 / 무료배송 여부
          • 옵션 수, 구매 건수
          • 평점, 리뷰 수
          📊
          가격/리뷰 변동 추적

          같은 상품을 여러 번 방문하면 가격과 리뷰 변화를 시간순으로 확인할 수 있습니다.

          🧮

          6. 마진 탭 — 마진 계산기

          ▼

          1688 원가 → 쿠팡 판매가 기준으로 예상 순이익과 마진율을 계산합니다.

          입력 항목

          1688 원가 (CNY)상품 원가 (위안)
          환율 (CNY→KRW)기본값 190
          국제배송비1건당 배송비 (원)
          관부가세율기본값 10%
          쿠팡 판매가예상 판매가 (원)
          쿠팡 수수료율카테고리별 (기본 10.8%)

          계산 결과

          원가(KRW), 배송비, 관부가세, 총 원가, 쿠팡 수수료, 예상 순이익, 마진율(%)

          💡 마진율이 초록색이면 이익, 빨간색이면 손해입니다.
          🔗

          7. 서버 탭 — 서버 연동

          ▼

          lumiriz.kr 계정으로 로그인하면 모든 데이터가 서버에 자동 동기화됩니다.

          ☁️
          동기화되는 데이터
          • 검색 스냅샷 (검색어, 상품 목록, 통계)
          • 소싱 후보 목록 및 상태
          • 순위 추적 데이터
          • 상품 상세 가격/리뷰 변동
          🖥
          웹 대시보드

          서버에 로그인하면 lumiriz.kr의 웹 대시보드에서 확장프로그램 통계를 확인할 수 있습니다.

          • 총 검색 횟수, 검색어 종류
          • 자주 검색한 키워드 TOP 10
          • 후보 상태별 현황 및 관리
          • 순위 추적 이력 (7/14/30일)
          🚀

          7
          웹 대시보드 관리
