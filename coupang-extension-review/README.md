# Coupang Search Helper MVP

이 샘플은 사용자가 직접 연 쿠팡 검색 결과 페이지에서 상품 카드를 읽어 Chrome 사이드패널에 표시하는 최소 기능 확장프로그램입니다.

## 포함 기능
- `https://www.coupang.com/np/search*` 페이지 자동 감지
- 검색 결과 상위 최대 20개 상품 파싱
- 상품ID, 제목, 가격, 평점, 리뷰 수, 링크 표시
- Chrome side panel에 결과 표시

## 설치
1. Chrome에서 `chrome://extensions` 열기
2. 우측 상단 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램을 로드합니다` 클릭
4. 이 폴더 선택
5. 쿠팡 검색 결과 페이지를 열고, 툴바 아이콘을 눌러 사이드패널 열기
